## 第 3 页：研究背景与临床痛点

```
标题：超声图像肿瘤诊断的三大挑战

痛点一：病灶边界模糊
超声图像信噪比低，肿瘤边缘灰度梯度微弱，
医师手动勾画主观性强，诊断一致性差

痛点二：小病灶漏检风险高
直径 < 1cm 的早期肿瘤在低分辨率超声中极易遗漏，
现有设备漏诊风险显著

痛点三：单一信息源局限
传统诊断仅依赖超声图像，忽略了 LV-GLS 等临床
指标的诊断价值

▌ 我国甲状腺癌过度诊断率已达 87%，基层漏诊风险更为突出
```

## 第 5 页：语义分割模型改进

```
标题：改进 YOLOv11 —— C3k2_DFF + IEMA

C3k2_DFF 双域动态融合模块
• 重构原始 C3K2 为并行双分支结构
• 通道注意力：抑制超声背景噪声
• 空间注意力：强化肿瘤边缘特征
• 1×1 卷积动态学习融合权重

IEMA 注意力机制（三阶段处理）
• 特征解耦：边缘支路 + 语义支路
• 特征增强：局部对比度增强 → 放大模糊边缘灰度梯度
• 特征融合：跨尺度动态权重

IEMA + DFF 协同：单尺度增强 + 跨尺度融合
```

## 第 8 页：TR-Net 图像分类

```
标题：TR-Net —— 超声图像二分类

设计思路
• 输入：语义分割提取的病灶区域图像
• 骨干网络：EfficientNet-B0
• 策略：ImageNet 预训练 + 迁移学习
• 优化：复合系数在深度、宽度、分辨率三维度自适应调整
• 核心模块：MBConv（深度可分离卷积 + SE 注意力）

训练 300 轮后精度稳定收敛于 85.3%
```

## 第 10 页：GDX 临床数据模型

```
标题：GDX 集成学习 —— 临床特征融合

改进一：基模型互补
XGBoost（稀疏特征 + 正则化） + GradientBoosting（连续特征 + 非线性）

改进二：堆叠集成
随机森林元分类器学习基模型权重 → 替代简单平均

改进三：自动化调参
GridSearchCV 系统搜索超参数空间

改进四：早停机制
验证集 AUC 连续 3 次不提升 → 终止训练，防过拟合

LV-GLS 与 LVEF 相关系数 r = 0.47 → 临床特征间存在关联，纯图像无法捕获
```

## 第 11 页：多维度融合策略

```
标题：多维度融合 —— 三个维度的协同互补

维度一：超声图像语义特征 → 病灶在哪里
维度二：病灶图像分类特征 → 病灶是什么
维度三：临床结构化数据     → 为什么是

         ↓ Stacking 堆叠集成 ↓

基学习器（TR-Net + GDX）→ 元分类器（随机森林）
→ 门控机制过滤噪声 → 最终诊断
```
